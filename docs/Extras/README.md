# Project Documentation and Extras

This folder contains supplementary project files organized by semester.

---

## Folder Structure

```
docs/Extras/
├── SD1_F25/          # Fall 2025 - Senior Design I
└── SD2_S26/          # Spring 2026 - Senior Design II
```

Each semester folder follows the same structure:

| Folder | Contents |
|--------|----------|
| `Bill-of-Materials-and-Budget/` | Purchase request forms, parts lists, and budget tracking spreadsheets |
| `Calculations/` | Engineering calculations and MATLAB live scripts (.mlx) |
| `Computer-Code/` | Firmware and application source code for the QPG6200 |
| `Correspondance/` | Email correspondence and meeting minutes |
| `Drawings/` | Electrical and mechanical drawings |
| `Pictures-Videos/` | Project photos and videos (files stored externally) |
| `Presentations/` | Design review presentations and meeting agendas |
| `Reports-Documents/` | Technical reports, contracts, specifications, and timesheets |
| `Research/` | Research documents organized by discipline (Computer, Electrical, Mechanical) |

---

## Semesters

### SD1 - Fall 2025
Senior Design I covered initial project research, component selection, PCB design, and prototype development. Key deliverables include the Preliminary Design Review (PDR) and Critical Design Review (CDR).

### SD2 - Spring 2026
Senior Design II covered PCB assembly and testing, firmware development, gateway integration, and final system demonstration. Key deliverables include the Progress Review Presentation (PRP), Project Status Review (PSR), and Final Report.
